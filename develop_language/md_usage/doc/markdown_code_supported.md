# MarkDown支持高亮显示的语言  

`Markdown`里对代码块的引用语法是三个撇{```code```},在其后可增加代码名称,比如`java`,`js`等标记该部分代码的类型.之后在页面展现的时候就可以高亮显示关键字了.
